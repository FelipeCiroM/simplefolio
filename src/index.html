Proyectos

Videojuegos

El proyecto consiste en una SPA (Single Page Application) que permite obtener información de una API que otorga la página rawg.io.
Van a encontrar un listado de videojuegos con una información detallada de cada uno al darle click en este.
Pueden crear sus propios videojuegos por medio de un formulario controlado. También pueden filtrarlos por género, creados o existentes, ordenarlos alfabéticamente, por rating y fecha de realización.
Las herramientas utilizadas fueron: JavaScript, HTML, CSS, NodeJS, Express, Sequelize, PostgreSQL, React y Redux.
